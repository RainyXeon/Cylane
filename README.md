## 📑 Short Feature

- [x] Music System
- [x] Multi Language
- [x] Playlist Network
- [x] SlashCommand
- [x] ContextMenus
- [x] Custom Filters
- [x] Play music from file
- [x] Easy to use
- [x] Autocomplete (Play command)
- [x] Auto restore lavalink (Get from [lavalink.darrennathanael.com](https://lavalink.darrennathanael.com/NoSSL/lavalink-without-ssl/) sources)
